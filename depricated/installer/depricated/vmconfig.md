# Island Virtual Machine configuration
This document contains all steps to re-create Island Virtual Machine

# Installation
Debian 9.5.0 amd64 version used as a base OS.


### Virtual hardware:
* Hard drive VMDK dynamically allocated 8GB
* RAM 1gb

### Install parameters
* hostname - island
* domain name - islands
* root pass - 123
* islands pass - 123
* Language - US Inglish

### Packages
* SSH server 
* system utilities
